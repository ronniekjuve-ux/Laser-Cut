# Fix: CacheManager button not working

## Problem
"Очистить кэш и перезагрузить" button is greyed out / doesn't work.
Root cause: when `navigator.serviceWorker.controller` is null, `clearCache()` clears caches but **never calls `window.location.reload()`** — so the page stays stuck.

## Fix
**File:** `frontend/src/components CacheManager.jsx`
- Add `window.location.reload()` after direct cache clear in the fallback path

## Verification
1. Rebuild frontend locally
2. Click ⚙️ → "Очистить кэш и перезагрузить" → page should reload
