# План: Группировка заказов

## Суть
Возможность объединять заявки от одного заказчика в группы (например, 8мм + 10мм + 12мм) и видеть информацию по группе целиком.

## Модель данных

### Новая таблица `order_groups`
| Поле | Тип | Описание |
|------|-----|----------|
| id | int PK | ID группы |
| name | String(100) | Название (опционально) |
| created_at | datetime | Дата создания |

### Поле `group_id` в `Application`
- `group_id: int | null` — FK → `order_groups.id`

## Бэкенд

### Новые эндпоинты (`app/api/v1/router_applications.py`)

1. **`POST /api/v1/applications/group`** — создать группу
   - Body: `{"app_ids": [1, 2, 3], "name": "Промстальмаш 16мм"}`
   - Создаёт `OrderGroup`, привязывает заявки
   - Возвращает `{"group_id": 1}`

2. **`DELETE /api/v1/applications/group/{group_id}`** — удалить группу
   - Отвязывает все заявки (set group_id = null), удаляет группу

3. **`PATCH /api/v1/applications/{app_id}/group`** — добавить/убрать заявку из группы
   - Body: `{"group_id": 5}` или `{"group_id": null}`

4. **`GET /api/v1/applications/group/{group_id}`** — детали группы
   - Все заявки группы + раскладки + итоги

### Обновление существующего `GET /api/v1/applications/`
- Добавить `group_id` и `group_name` в ответ каждого элемента

## Фронтенд

### Таблицы (`OrdersList.jsx`, `ApplicationsList.jsx`)
- Новый столбец **«Группа»** (после «Заказчик»)
- Показывает название группы или ID
- Сортировка и фильтр по группе
- Клик на название группы → открытие карточки группы

### Управление группами
- Контекстное меню (правый клик) или чекбоксы → кнопка «Создать группу» / «Добавить в группу»
- Модалка создания группы: ввод названия, подтверждение

### Карточка группы (модальное окно)
- Заголовок: название группы
- Список заявок группы:
  - ID, заказчик, материал, толщина, статус, станок
  - Прогресс резки (если есть раскладки)
- Итоги: общее количество заявок, общая масса, общее кол-во деталей
- Кнопка «Отвязать заявку» (убрать из группы)
- Кнопка «Удалить группу»

## Верификация
1. Создать группу из 2-3 заявок через UI
2. Проверить что столбец «Группа» отображается в таблицах
3. Открыть карточку группы — проверить что все заявки отображаются
4. Отвязать заявку из группы — проверить что она исчезает из группы
5. Удалить группу — проверить что заявки остаются
