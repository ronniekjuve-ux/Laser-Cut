# План: Исправление качества изображений раскладок

## Корневая причина

Документы раскладок генерируются программой CypCut и сохраняются как `.DOC` файлы. Изображения раскладки (контуры деталей на листе) встроены в DOC как **векторные EMF/WMF** файлы.

Текущий процесс извлечения (`unified_parser.py:643-728`):
1. `libreoffice --headless --convert-to html` — конвертирует DOC→HTML
2. LibreOffice **растеризует** векторные EMF/WMF изображения в PNG/GIF/JPG
3. Растеризация происходит с **низким DPI** (по умолчанию ~96 DPI)
4. Кривые линии упрощаются в прямые отрезки, мелкие детали теряются

Результат: на скриншоте видно, что в оригинале раскладка показывает плавные кривые, а в приложении — ломаные прямые с пропавшими сегментами.

## Решение: DOC → PDF → PNG

Вместо `DOC→HTML` использовать **`DOC→PDF→PNG`**:
- PDF-экспорт LibreOffice **сохраняет векторные данные** гораздо лучше
- Рендеринг PDF→PNG через `poppler`/`pdf2image` позволяет задать **высокое разрешение** (300 DPI)

### Почему PDF лучше HTML:
- HTML-экспорт LibreOffice конвертирует EMF→растровое изображение сразу (потеря качества)
- PDF-экспорт сохраняет векторные пути, текст, шрифты в исходном качестве
- Финальный рендеринг PDF→PNG контролирует DPI на выходе

### 2. Модифицировать `extract_images()`
**Файл:** `app/services/unified_parser.py` (строки 643-728)

Новая логика:
1. Конвертировать DOC→PDF через `libreoffice --headless --convert-to pdf`
2. Рендерить PDF→PNG через `pdf2image` с 300 DPI
3. Сохранять PNG файлы в выходную директорию
4. Fallback на старый метод (HTML→images) если PDF-конвертация не удалась

```python
def extract_images(filepath: str, output_dir: str, prefix: str = "", filter_dft: bool = False) -> List:
    import tempfile, subprocess, shutil
    from pathlib import Path

    saved = []

    try:
        with tempfile.TemporaryDirectory() as tmpdir:
            # Шаг 1: DOC → PDF (сохраняет векторное качество)
            result = subprocess.run(
                ['libreoffice', '--headless', '--convert-to', 'pdf', '--outdir', tmpdir, filepath],
                capture_output=True, timeout=120
            )

            pdf_files = list(Path(tmpdir).glob('*.pdf'))

            if pdf_files:
                # Шаг 2: PDF → PNG (высокое разрешение)
                try:
                    from pdf2image import convert_from_path
                    images = convert_from_path(
                        str(pdf_files[0]),
                        dpi=300,  # Высокое разрешение
                        fmt='png',
                        output_folder=tmpdir,
                        output_file='layout_'
                    )
                    # Копируем изображения
                    dest_dir = Path(output_dir) / prefix
                    dest_dir.mkdir(parents=True, exist_ok=True)
                    for i, img_path in enumerate(Path(tmpdir).glob('layout_*.png')):
                        dest_path = dest_dir / f"layout_{i+1}.png"
                        shutil.copy2(img_path, dest_path)
                        saved.append(f"/api/v1/images/{prefix}/layout_{i+1}.png")
                except ImportError:
                    # Fallback: если pdf2image не установлен
                    pass

            # Fallback: старый метод через HTML (если PDF не сработал)
            if not saved:
                # ... существующий код через HTML ...
                pass

    except Exception as e:
        print(f"Warning: image extraction failed: {e}")

    return saved
```
