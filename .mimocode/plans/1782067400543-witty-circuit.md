# Исправление опечатки: просчёт → расчёт

## Проблема
На вкладке "Заявки" отображается текст "Предварительный просчёт" вместо "Предварительный расчёт". Опечатка встречается в 3 файлах (4 вхождения).

## Файлы для исправления

### 1. `frontend/src/components/Layout.jsx`
- Строка 102: `if (p === '/') return 'Предварительный просчёт';`
- Строка 112: `return 'Предварительный просчёт';`

### 2. `frontend/src/pages/Applications/ApplicationsList.jsx`
- Строка 82: `<h3>Предварительный просчёт — {app.order_name || app.id}</h3>`

### 3. `frontend/src/pages/Applications/CostCalculator.jsx`
- Строка 36: `<div ...>Предварительный просчёт</div>`

## Изменение
Заменить `просчёт` → `расчёт` во всех 4 местах.

## Верификация
- Запустить фронтенд (`npm run dev` в `frontend/`)
- Проверить, что на главной странице и в модальном окне заявки отображается "Предварительный расчёт"
